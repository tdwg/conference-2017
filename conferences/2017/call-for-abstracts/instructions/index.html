Instructions for Submitting Abstracts

  1. Log in at http://tdwgproceedings.pensoft.net.
  2. Click the “Start a Conference Abstract” button to directly open an abstract manuscript.
  3. Click “Collections” on the top navigation bar. This is where you select the conference track to which you are submitting. If you are an invited speaker for a symposium or workshop, select that track. If you are submitting an oral presentation for consideration by an open symposium or workshop, choose that track (this does not guarantee acceptance in that track). (See https://tdwg.github.io/conferences/2017/program/ for descriptions and indications of which track or open to unsolicited submissions.) If you would like to give an oral presentation, and no existing track is appropriate, choose “Other Oral Presentations”. You can also select “Poster” or “Computer demonstration”.
